Quick Start

Get up and running in 5 minutes 

# 1. Clone the repository
git clone https://github.com/JamesTheGiblet/adaptive-intelligence.git
cd adaptive-intelligence

# 2. Configure environment
cp .env.example .env
# Add your API keys (GEMINI_API_KEY=your_key_here)

# 3. Start the platform
docker-compose up -d

# 4. Verify it's working
            

            


            

                🎉 You're now running: Praximous Gateway (port 8000), AAS Agents (port 9000), and all Intelligence Tools!
            


            

                


# Try scanning for secrets
curl -X POST http://localhost:8000/api/v1/process \
  -d '{"task_type": "whisper_scan", "path": "."}'

                

                
                


# Generate tests automatically
curl -X POST http://localhost:8000/api/v1/process \
  -d '{"task_type": "generate_tests", "code": "def add(a,b): return a+b"}'
